# Day 15 - PM - Database Theory and Random Variables

## Part A:- Concept Application 

This section applies database design concepts to an online food delivery application scenario. It covers designing an ER diagram, normalizing a dataset into 1NF, 2NF, and 3NF, writing relational algebra queries, and mapping Pandas operations to equivalent relational algebra operations.


## Key Learnings :- 

- Understood how to model a real-world application using ER design principles with entities, attributes, relationships, and cardinality.

- Learned how normalization reduces redundancy and improves consistency through 1NF, 2NF, and 3NF.

- Connected practical Pandas operations with relational algebra concepts used in database systems.

- Improved understanding of how conceptual design, logical design, and query operations work together in relational databases.
