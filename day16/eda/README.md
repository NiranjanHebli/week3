# Day 16 - PM - EDA report

## Part A:- Concept Application 

This EDA report examines a dataset containing 1200 records and 10 mixed-type variables, aiming to understand data distribution, detect relationships, and identify correlations.


## Key Learnings

- Numerical variables may show normal, skewed, or multimodal distributions

- Regression plots help detect positive, negative, or weak relationships

- Correlation heatmaps reveal strong and weak dependencies

- Box plots identify median differences, spread, and outliers across categories
