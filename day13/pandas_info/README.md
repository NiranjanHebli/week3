In this section, we will explore Pandas, powerful data manipulation capabilities by creating a comparison report for sales data from 3 different months.

### Key Learnings

- Calculating metrics (total revenue, average order value, top-selling product) for each month.
- Creating a summary comparison DataFrame with months as index and metrics as columns.
